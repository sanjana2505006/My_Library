A modern, full-stack library management application built with React, Node.js, and Express.

## Features

-   **Browse & Discover**: Explore a curated collection of Romance, Thriller, Fiction, and Fantasy novels.
-   **Read & Download**: Read books directly in the browser or download them for offline enjoyment.
-   **User Accounts**: Sign up and login to manage your profile (Auth protected).
-   **Upload Books**: Contribute to the library by uploading your own books (EPUB/PDF/Text).
-   **Dark Mode**: A beautiful, eye-friendly dark theme (and light mode!).
-   **Responsive Design**: Works perfectly on desktop, tablet, and mobile.

## Tech Stack

-   **Frontend**: React (Vite), React Router, Context API, CSS Modules (Custom Styling).
-   **Backend**: Node.js, Express, File System (JSON DB).
-   **Design**: Custom CSS with a premium, glassmorphism-inspired aesthetic.

## Getting Started

1.  **Server**:
    ```bash
    cd library-app/server
    npm install
    npm start
    ```

2.  **Client**:
    ```bash
    cd library-app/client
    npm install
    npm run dev
    ```
